## About The Project
A decentralized application where an investor deposits a certain number of [DAI](https://makerdao.com/en/) coins and gets back extra coins as a reward for investing on the platform.

## Tech stack and libraries
 - Solidity
 - Ethereum Blockchain
 - Metamask
 - React
 - web3.js
 - Bootstrap

## Getting Started
